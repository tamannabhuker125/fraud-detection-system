#Fraud Detection System 

## Overview 
This project predicts whether a financial transaction is Fraud or Legitimate using a Machine Learning model

## Features
-Predicts fraudulent transaction
-User-friendly Streamlit interface
-Fast real-time prediction

## Technologies
-Python
-Streamlit
-Pandas
-Scikit-learn
-joblib

## How to run
1. install the required libraries:
   pip install -r
   requirements.txt

2. Run the application:
   streamlit run
   fraud_detection.py
